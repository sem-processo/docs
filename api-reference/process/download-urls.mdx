## Endpoint

`POST /process/download-urls`

Receba um array de IDs de anexos e obtenha URLs temporárias para download direto dos arquivos armazenados.

<Note>
  As URLs geradas expiram em **30 minutos**. Após esse período, será necessário
  fazer uma nova requisição para obter URLs atualizadas.
</Note>

## Exemplo de requisição

```bash
curl -X POST https://api.lexxy.com.br/process/download-urls \
  -H "Content-Type: application/json" \
  -H "x-api-key: SUA_API_KEY" \
  -d '{
    "ids": [
      "attachment_uuid_v4",
      "attachment_uuid_v4"
    ]
  }'
```

```json
{
  "encontrados": [
    {
      "id": "attachment_uuid_v4",
```

## Download automático com ZIP

Se você precisar baixar múltiplos arquivos de uma vez, o exemplo abaixo
busca as URLs na API e já compacta tudo em um `.zip` localmente —
**os arquivos nunca passam pelo seu servidor**, vão direto do S3 para o zip.

<CodeGroup>

```javascript Node.js
import fetch from "node-fetch";
import archiver from "archiver";
import fs from "fs";

const API_URL = "https://api.lexxy.com.br/process/download-urls";
const API_KEY = "SUA_API_KEY";

async function downloadAndZip(ids, outputZipPath = "arquivos.zip") {
  // 1. Busca as URLs na API
  const response = await fetch(API_URL, {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "x-api-key": API_KEY,
    },
    body: JSON.stringify({ ids }),
  });

  const { encontrados, nao_encontrados } = await response.json();

  if (nao_encontrados.length > 0) {
    console.warn("IDs não encontrados:", nao_encontrados);
  }

  // 2. Cria o zip
  const output = fs.createWriteStream(outputZipPath);
  const archive = archiver("zip", { zlib: { level: 6 } });

  archive.pipe(output);

  // 3. Baixa cada arquivo direto do S3 via stream e adiciona ao zip
  await Promise.all(
    encontrados.map(async ({ nome_arquivo, download }) => {
      const fileResponse = await fetch(download.url);

      if (!fileResponse.ok) {
        console.error(`Falhou ao baixar: ${nome_arquivo}`);
        return;
      }

      archive.append(fileResponse.body, { name: nome_arquivo });
    })
  );

  // 4. Finaliza e salva
  await archive.finalize();

  await new Promise((resolve, reject) => {
    output.on("close", resolve);
    output.on("error", reject);
  });

  console.log(`ZIP gerado: ${outputZipPath} (${archive.pointer()} bytes)`);
}

// Uso
downloadAndZip([
  "attachment_uuid_v4",
  "attachment_uuid_v4",
], "./anexos-processo.zip");
```

```python Python
import requests
import zipfile
import io
import os

API_URL = "https://api.lexxy.com.br/process/download-urls"
API_KEY = "SUA_API_KEY"

def download_and_zip(ids, output_zip_path="arquivos.zip"):
    # 1. Busca as URLs na API
    response = requests.post(
        API_URL,
        headers={"Content-Type": "application/json", "x-api-key": API_KEY},
        json={"ids": ids},
    )
    data = response.json()

    encontrados = data.get("encontrados", [])
    nao_encontrados = data.get("nao_encontrados", [])

    if nao_encontrados:
        print(f"IDs não encontrados: {nao_encontrados}")

    # 2. Baixa e compacta cada arquivo
    with zipfile.ZipFile(output_zip_path, "w", zipfile.ZIP_DEFLATED) as zipf:
        for item in encontrados:
            nome_arquivo = item["nome_arquivo"]
            url = item["download"]["url"]

            file_response = requests.get(url, stream=True)

            if not file_response.ok:
                print(f"Falhou ao baixar: {nome_arquivo}")
                continue

            # Stream direto para o zip sem salvar em disco
            with zipf.open(nome_arquivo, "w") as zf:
                for chunk in file_response.iter_content(chunk_size=8192):
                    zf.write(chunk)

            print(f" Adicionado: {nome_arquivo}")

    print(f"ZIP gerado: {output_zip_path}")

# Uso
download_and_zip([
    "attachment_uuid_v4",
    "attachment_uuid_v4",
], "./anexos-processo.zip")
```
